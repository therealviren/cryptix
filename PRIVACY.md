# Cryptix Coin Bot Privacy Policy

**Last Updated:** January 5, 2026

## 1. Introduction
Cryptix Coin Bot ("we", "our", "us") values your privacy. This Privacy Policy explains how we collect, store, and use information from users of the Bot.

By using the Bot, you consent to the practices described herein.

---

## 2. Data Collection
We only collect data necessary for the Bot's functionality:
- Discord User ID.
- Coin balances.
- Redeemed codes.
- Blocked user status.
- Admin actions for moderation logs.
No other personal information is collected unless voluntarily provided.

---

## 3. Data Usage
- User IDs and coin balances are used to manage virtual currency.
- Admin logs track moderation actions (blocking users, code generation) to prevent abuse.
- Redeemed codes are stored temporarily to validate usage and are deleted after redemption.

---

## 4. Data Storage
- Data is stored securely in a local SQLite database.
- We use reasonable technical measures to prevent unauthorized access.
- Data is only accessed by the Owner and authorized Admins for management purposes.

---

## 5. Data Sharing
- We **do not sell, rent, or share your personal data** with third parties.
- Data may only be shared when required by law.

---

## 6. User Rights
- Users may request deletion or blocking of their data via contacting the Owner.
- Coin balances and code redemption data are considered part of Bot functionality and may not be fully recoverable after deletion.

---

## 7. Security
- We use secure coding practices and logs to prevent abuse.
- No system is fully secure; using the Bot carries inherent risks.
- In case of a security breach, affected users will be informed promptly.

---

## 8. Third-party Services
- The Bot runs on Discord and relies on Discord APIs.
- Discord may collect additional data as outlined in their own Privacy Policy.

---

## 9. Changes to Privacy Policy
- We may update this Privacy Policy at any time.
- Users will be bound by changes upon continued use of the Bot.
